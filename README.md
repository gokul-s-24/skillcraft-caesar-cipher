# Caesar Cipher Tool (Python GUI)

A modern Caesar Cipher encryption and decryption tool built using Python with an interactive graphical user interface.

This project demonstrates basic cryptography concepts along with a user-friendly UI for better usability.

---

## Features

* Encrypt and Decrypt messages
* Modern Dark Mode UI (CustomTkinter)
* Simple and easy-to-use interface
* Supports uppercase and lowercase letters
* Copy encrypted/decrypted result to clipboard
* Real-time processing
* Placeholder input behavior (auto disappears while typing)

---

## Tech Stack

* Python
* Tkinter / CustomTkinter

## Installation

1. Clone the repository:

```bash
git clone https://github.com/gokul-s-24/skillcraft-caesar-cipher.git
cd skillcraft-caesar-cipher
```

2. Install required library:

```bash
pip install customtkinter
```

3. Run the application:

```bash
python caesar_cipher.py
```

---

## How It Works

The Caesar Cipher shifts each letter in the message by a fixed number (shift value).

Example:
Message: HELLO
Shift: 3
Output: KHOOR
